Authors:

Developed a Contacts Application using the Windows Presentation Foundation on Windows Phone. This application allows users to create dynamic fields associated with Contacts, create and assign groups, schedule events and send SMS/Email messages to Contacts and Groups of Contacts.

The project was developed using C# and XAML, using JSON as a persistence mechanism. The goal of this project was to help develop our abilities in Mobile Application Development, we wanted to gain a knowledge in C# and other Microsoft Technologies.
